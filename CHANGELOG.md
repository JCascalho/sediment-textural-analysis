# Changelog

## Version 2.0.0

### Added

- EDUCOAST educational grain-size database: `Data_base_EDUCOAST.xlsx`.
- Integrated graphical and moments parameter-calculation routine.
- Separate plotting routine for educational workflow transparency.
- Automatic default processing of the EDUCOAST workbook.
- Output workbook prepared for plotting and STA-related workflows.
- Probability-paper and cumulative grain-size curve plotting.
- Frequency, histogram, scatter, and bivariate plotting options.

### Changed

- Expanded the repository from a Folk and Ward graphical routine to an EDUCOAST educational workflow.
- Kept calculation and plotting routines separate for pedagogical reasons.
- Updated default filenames to match the EDUCOAST dataset.
- Reduced probability-axis Y tick label size in plotting outputs.

### Fixed

- Removed personal local paths from public routines.
- Standardized public defaults for GitHub, Zenodo, and Spyder use.
