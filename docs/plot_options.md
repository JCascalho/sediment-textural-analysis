# Plot Options

Use `--plot` to choose which figure type to generate.

Available options:

- `probability-paper`
- `cumulative`
- `frequency`
- `histogram`
- `cumulative-linear`
- `excel-style`
- `scatter`
- `bivariate`
- `bivariate-fields`
- `all`

Examples:

```bash
python sediment_textural_graphical_moments_plots.py --plot probability-paper
python sediment_textural_graphical_moments_plots.py --plot cumulative
python sediment_textural_graphical_moments_plots.py --plot bivariate
python sediment_textural_graphical_moments_plots.py --plot all
```
